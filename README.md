# replit-builds

Production-ready websites and apps shipped from Replit. Each top-level folder is a self-contained project with its own workspace.

## Projects

| Folder | Project | Status |
|---|---|---|
| [`hope-harbor/`](./hope-harbor) | **Hope Harbor** — Addiction treatment navigation brand. Concept lab exploring five premium homepage directions. | Active |

## Conventions

- Each project lives in its own top-level folder named after the brand or product (e.g., `hope-harbor/`).
- Inside each project folder is the full pnpm monorepo workspace as built on Replit.
- New Replit builds get their own folder — never co-mingle codebases.
